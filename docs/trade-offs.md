# Trade-offs

- Stream real vs batch simplificado: streaming para menor latencia
- Threshold fixo vs dinamico: fixo no MVP, dinamico no futuro
- Complexidade de regras vs performance: regras limitadas para p95
- Bloqueio automatico vs manual review: bloqueio automatico para scores altos
- Persistencia completa vs eventos criticos: persistir alertas e eventos relevantes
